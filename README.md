# This repository is for the paper 'SANGRE: a Shallow Attention Network Guided by Resolution Expansion for MR Image Segmentation'

## Data preparation
* ACDC - The data is available from [MICCAI2017 challenge](https://www.creatis.insa-lyon.fr/Challenge/acdc/miccai_results.html), or you can download the processed dataset from Google Drive of [MT-UNet](https://drive.google.com/file/d/13qYHNIWTIBzwyFgScORL2RFd002vrPF2/view) and save the data to './data/ACDC/' folder'.
